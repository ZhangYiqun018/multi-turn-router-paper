\textbf{Strategy for HLE Questions}
\begin{enumerate}[leftmargin=*, label=\arabic*.]
\item \textbf{Understand the question}: read carefully and identify what's being asked
\item \textbf{Break down the problem}: decompose into sub-problems if needed
\item \textbf{Research if needed}: use \texttt{search} for factual information you're unsure about
\item \textbf{Verify sources}: use \texttt{browse} to read primary sources when accuracy matters
\item \textbf{Compute when needed}: use \texttt{python} for calculations and data analysis
\item \textbf{Synthesize}: combine information from multiple sources
\item \textbf{Verify your answer}: double-check before submitting
\item \textbf{Submit}: use \texttt{answer} with your final answer
\end{enumerate}

\vspace{0.2cm}
\textbf{Important Notes}
\begin{itemize}
\item These are challenging questions---take your time
\item Be precise---exact answers are often required
\item Show your reasoning before using tools
\item If uncertain, express confidence level in your answer
\end{itemize}
